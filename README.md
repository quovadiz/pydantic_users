Необходимо реализовать иерархию моделей пользователей с помощью Pydantic 2:

1. **`BaseUser`** – базовая модель с общими полями (email, имя, фамилия).

2. **`User`** – обычный пользователь (наследуется от `BaseUser`), добавляет **пароль** и **возраст**.

3. **`AdminUser`** – админ (наследуется от `User`), добавляет **уровень доступа (role)** и **метод проверки прав**.

**Требования:**
✅ Валидация email (должен быть корректным). \
✅ Имя и фамилия – не пустые, если передана строка в нижнем регистре, первая буква автоматически должна стать заглавной (петя → Петя, ВаСЯ → Вася). \
✅ Пароль: минимум 8 символов, хотя бы 1 цифра и 1 спецсимвол (`!@#$%^&*`). \
✅ Возраст ≥ 18. \
✅ У админа `role` может быть только `["admin", "superadmin"]`. \
✅ У админа должен быть метод `has_permission(permission: str) -> bool`. \

**Конечный результат:**

- Файл с моделями

- Файл с тестами на создание моделей и проверки всех требований (к тестированию подходить ответственно, со всеми знаниями о тест дизайне!)
